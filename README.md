# Marco Scarpelli - Academic Portfolio

This is a modern, high-performance academic portfolio website built with React, TypeScript, and Tailwind CSS. It features a sleek dark theme with smooth animations and a responsive design.

## Features

- **Modern UI/UX**: Dark-themed design with a focus on typography and readability.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Smooth Animations**: Powered by `motion/react` for a premium feel.
- **Dynamic Content**: Easily customizable via `src/constants.ts`.
- **Lucide Icons**: Clean and consistent iconography.

## Local Setup

To run this project locally, follow these steps:

1. **Clone or Download**: Get the project files on your local machine.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Add Your Hero Image**:
   - Place your personal photo in the `public/` directory.
   - Rename it to `marcoscarpelli.jpeg` (or update the path in `src/App.tsx`).
4. **Run Development Server**:
   ```bash
   npm run dev
   ```
5. **Build for Production**:
   ```bash
   npm run build
   ```

## Technologies Used

- **React 18**: Frontend library.
- **TypeScript**: For type safety and better developer experience.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Motion (Framer Motion)**: For advanced animations.
- **Lucide React**: Icon library.
- **Vite**: Fast build tool and development server.

## Customization

Most of the personal information, projects, and experience can be updated in:
`src/constants.ts`

Enjoy your new portfolio!
